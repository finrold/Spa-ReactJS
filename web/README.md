### Como usar
Primeiro baixe o projeto para seu computador como o comando:
```sh
$ git clone git@github.com:finrold/SpaReactImobiliaria.git
```

Depois de baixar abre o terminal e digite o caminho onde está a pasta web.
#### Startando o Projeto:
```sh
$ cd /web
$ npm start
```
